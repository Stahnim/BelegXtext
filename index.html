Willkommen bei meiner Eclipse Update Site!

Diese Update-Site enthält Plug-ins und Features, die in Eclipse installiert werden können.

Installation

Um diese Update-Site in Eclipse hinzuzufügen, führen Sie die folgenden Schritte aus:

  1. Öffnen Sie Eclipse.
  2. Gehen Sie zu Help → Install New Software....
  3. Klicken Sie auf Add....
  4. Geben Sie die folgende URL ein:
    https://.github.io/eclipse-update-site/
  5. Wählen Sie die gewünschten Plug-ins und Features aus und klicken Sie auf Install.

Vielen Dank, dass Sie diese Update-Site nutzen!
